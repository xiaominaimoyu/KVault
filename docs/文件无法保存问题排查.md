## **排查一：数据目录路径是否固定（最可能）**

## **排查五：closeEvent 是否过早终止写入线程**

README 提到"关闭窗口时会自动终止所有后台工作线程"。如果用户导入文件后立即关闭程序，而后台的向量化 Worker 线程尚未完成写入就被强制终止，数据可能只写了一部分或完全没写入。

**检查方法**：打开 `gui/main_window.py` 中的 `closeEvent` 方法，确认：

```python
def closeEvent(self, event):
    # 确保所有 Worker 线程完成后再关闭
    for worker in self.active_workers:
        worker.wait(5000)  # 最多等待 5 秒
    # 关闭数据库连接
    self.metadata_manager.close()
    event.accept()
```

如果使用的是 `worker.terminate()`（强制终止），改为 `worker.wait()`（等待完成）更为安全。
